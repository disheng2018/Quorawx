# 第2种

```Python
for i in range(1,10):
    for j in range(1,i+1):
        print("%d*%d=%2d" % (j,i,i*j),end=" ")
    print (" ")
```

# 第5种

```Python
for i in range(1,10):
    for j in range(1,i):
        print(end='       ')
    for j in range(i,10):
        print('%d*%d=%2d' % (i,j,i*j),end=' ')
    print('')
```
